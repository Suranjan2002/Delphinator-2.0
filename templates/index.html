About

What Are We

Delphinator is a learning platform that provides, people who are about to step into the domain of Cryptography or are new to it, an opportunity to:

  • Get a boost and enthusiasm in unraveling the mysteries of cryptography.
  • Comprehend how the cipher and hash algortihms work with flowcharts and built-in calculators for simulated experience.
  • Learn about the unknown facts about cryptography and their applications in reality.

This website is developed by an aggregation of 4 sophomores from the CSE commune of the halls of ivy, Techno International New Town. This website also provides various features for better user experience and opportunity to explore new horizons.

Features

Check our features

Combi-pher

Tool for multi-cipher decrypt/encrypt Caesar Cipher, Vignere Cipher and Rail Fence Cipher for enhanced security.

ROT13 Cipher

Tool to decrypt/encrypt with ROT13. In ROT13 cipher, the shift is equal to 13, this allow the cipher to be reciprocal

Base Cipher

Tool to write numbers in base N. A base (radix) is the value of successive powers when writing a number.

RSA Cipher

Tool to decrypt/encrypt with RSA cipher. RSA is an asymmetric algorithm for data exchange over the Internet.

Morse Code

Tool to decrypt/encrypt Morse code. Morse code is a system allowing to code a message by short and long impulsion.

MD5 Hash

Tool to decrypt/encrypt MD5 function automatically, decryption via a huge dictionary of potential passwords.
